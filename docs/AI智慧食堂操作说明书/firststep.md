# 系统模型

为了更加好的适应当前网络及社会发展的主流，将传统的食堂模式移植到目前流行的互联网思路上来，数鸿智慧食堂就是在这样的一种背景下应运而生。该平台将借助互联网的优势，结合线下资源，共同打造一个线上为主，线下为辅的共赢平台。  

系统的主要功能需求如下：
